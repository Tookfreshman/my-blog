# my-blog

> This is a web development project.

## Build Setup

```bash
# install dependencies
yarn install

# serve with hot reload at http://localhost:3000
yarn start

# build for production with minification
yarn build

```

#### vue-third-legal 项目架构

```
├── dist                     # 打包生成文件
├── public
│   ├── favicon.ico          # Favicon
│   └── index.html
├── src
│   ├── assets               # 本地静态资源
│   ├── components           # 业务通用组件
│   ├── lang                 # 语言翻译
│   ├── views                # 业务页面
│   ├── router               # 路由管理
│   ├── services             # 后台接口服务
│   ├── store                # vuex状态管理
│   ├── styles               # 样式库
│   ├── utils                # 工具库
│   ├── App.vue              # 全局入口视图页面
│   └── main.js              # 全局 JS
│
├── README.md
├── .eslintrc.js
└── .gitignore
```
